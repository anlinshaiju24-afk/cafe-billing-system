# Coffee & Crumbs Cafe Billing System

An end-to-end command-line cafe ordering and billing application developed in Python. This project simulates a real-world 
Point of Sale (POS) system by allowing customers to browse a categorized menu, place multiple orders, calculate bills
automatically, apply taxes and discounts, and generate an itemized receipt.

---

### System Architecture and Workflow

* Menu Management: Organizes food and beverage items into categorized dictionaries for efficient lookup and management.
* Order Processing: Accepts customer orders, validates item availability, records quantities, and supports multiple item purchases within a single session.
* Billing Engine: Calculates item-wise costs, subtotal, applicable taxes, discount eligibility, and the final payable amount.
* Receipt Generation: Produces a structured receipt displaying ordered items, quantities, pricing, tax, discount, and grand total.
* Order Simulation: Simulates order preparation using Python's `time` module before notifying the customer that the order is ready.

---

### Tech Stack

* Programming Language: Python
* Standard Library: Time Module

* Core Concepts: Loops, Conditional Statements, User Input Validation, Functions, Arithmetic Operations

### Application Features

* Categorized cafe menu
* Multiple item ordering
* Quantity validation
* Automatic subtotal calculation
* 5% tax calculation
* 10% discount on orders above ₹500
* Receipt generation
* Command-line interface
* Order preparation simulation

---

### Project Workflow

1. Display the complete cafe menu.
2. Accept customer orders.
3. Validate item availability.
4. Record quantity for each ordered item.
5. Calculate item-wise and overall billing.
6. Apply tax and discount rules.
7. Generate the final receipt.
8. Simulate order preparation.
9. Notify the customer when the order is ready.
